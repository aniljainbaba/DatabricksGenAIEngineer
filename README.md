# Databricks GenAI Engineer Training

A comprehensive training project covering Databricks fundamentals, Generative AI, LangChain, and Retrieval Augmented Generation (RAG) implementations.

## Project Structure

### 📚 Databricks_Fundamentals/
Introduction to Databricks platform and data processing:
- **Module_1.ipynb** - Getting started with Databricks
- **Module_2.ipynb** - Data processing and transformation
- **Module_3.ipynb** - Advanced data operations
- **Module_4.ipynb** - Best practices and optimization
- CSV datasets from 2019-2021 for practice exercises

### 🤖 GenAI_Fundamentals/
Generative AI and Large Language Model fundamentals:
- **Chat_Application.ipynb** - Building conversational AI applications
- **LLM_API_DB_SDK.ipynb** - Integrating LLM APIs with Databricks
- **SQL_AI_Query.ipynb** - AI-powered SQL query generation
- **openai_api.ipynb** - Working with OpenAI APIs
- Restaurant reviews dataset for sentiment analysis and demonstrations

### 🔗 LangChain/
LangChain framework implementation and integration:
- **Getting_Started_with_LangChain.ipynb** - Introduction to LangChain
- **SequentialChain.ipynb** - Building sequential chains and workflows
- **Unity_Catalog_Hosted_SQL_Function.ipynb** - Integration with Databricks Unity Catalog
- Electronics products dataset for practical examples

### 🎯 RAG/
Retrieval Augmented Generation implementation:
- **Data_Prep.ipynb** - Preparing documents for RAG
- **Vector_Embeddings.ipynb** - Creating and managing vector embeddings
- **Vector_Index_Creation.ipynb** - Building vector indices
- **RAG_Application.ipynb** - Building RAG applications
- **RAG_Evaluation.ipynb** - Evaluating RAG system performance
- Knowledge base with documents and images for RAG demonstrations

## Getting Started

1. Ensure you have Databricks workspace access
2. Clone or download this repository
3. Import the Jupyter notebooks into your Databricks workspace
4. Configure necessary API keys (OpenAI, etc.) as needed
5. Start with Databricks_Fundamentals modules and progress through each section

## Prerequisites

- Databricks workspace account
- Python 3.8+
- Required libraries: pandas, numpy, langchain, openai, databricks-sdk
- API keys for OpenAI or other LLM services (where applicable)

## Topics Covered

- ✅ Databricks fundamentals and data operations
- ✅ Large Language Models (LLMs) and APIs
- ✅ Generative AI applications
- ✅ LangChain framework and chains
- ✅ Vector embeddings and similarity search
- ✅ Retrieval Augmented Generation (RAG)
- ✅ Integration with Unity Catalog
